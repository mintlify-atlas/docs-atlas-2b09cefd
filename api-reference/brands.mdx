## List Brands

<api method="GET" path="/api/brands" auth="required">
  List all brands belonging to the authenticated user with their latest visibility scores.
</api>

### Response

<ResponseField name="brands" type="Brand[]">
  Array of brand objects
  
  <Expandable title="Brand properties">
    <ResponseField name="id" type="string" required>
      Unique brand identifier
    </ResponseField>
    
    <ResponseField name="user_id" type="string" required>
      Owner's user ID
    </ResponseField>
    
    <ResponseField name="name" type="string" required>
      Brand name
    </ResponseField>
    
    <ResponseField name="website_url" type="string" required>
      Brand website URL
    </ResponseField>
    
    <ResponseField name="industry" type="string | null">
      Brand industry (SaaS, E-commerce, Finance, Healthcare, Agency, Other)
    </ResponseField>
    
    <ResponseField name="is_active" type="boolean" required>
      Whether the brand is active
    </ResponseField>
    
    <ResponseField name="created_at" type="string" required>
      ISO 8601 timestamp
    </ResponseField>
    
    <ResponseField name="updated_at" type="string" required>
      ISO 8601 timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl -X GET "https://api.opensight.ai/api/brands" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Example Response

```json
{
  "brands": [
    {
      "id": "brand_123",
      "user_id": "user_456",
      "name": "Acme Inc",
      "website_url": "https://acme.com",
      "industry": "SaaS",
      "is_active": true,
      "created_at": "2024-01-15T10:00:00Z",
      "updated_at": "2024-01-15T10:00:00Z"
    }
  ]
}
```

## Create Brand

<api method="POST" path="/api/brands" auth="required">
  Create a new brand. Plan limits apply based on your subscription tier.
</api>

### Request Body

<ParamField body="name" type="string" required>
  Brand name (1-255 characters)
</ParamField>

<ParamField body="website_url" type="string" required>
  Valid brand website URL
</ParamField>

<ParamField body="industry" type="string">
  Brand industry (SaaS, E-commerce, Finance, Healthcare, Agency, Other)
</ParamField>

### Response

<ResponseField name="brand" type="Brand">
  The created brand object
</ResponseField>

### Example Request

```bash
curl -X POST "https://api.opensight.ai/api/brands" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Inc",
    "website_url": "https://acme.com",
    "industry": "SaaS"
  }'
```

### Example Response

```json
{
  "brand": {
    "id": "brand_123",
    "user_id": "user_456",
    "name": "Acme Inc",
    "website_url": "https://acme.com",
    "industry": "SaaS",
    "is_active": true,
    "created_at": "2024-01-15T10:00:00Z",
    "updated_at": "2024-01-15T10:00:00Z"
  }
}
```

### Errors

- **403 Forbidden** - Brand limit reached for your plan
- **400 Bad Request** - Invalid input data

## Get Brand Dashboard

<api method="GET" path="/api/brands/:id/dashboard" auth="required">
  Get aggregated dashboard data including overall score, engine scores, sentiment, mentions, and recent changes.
</api>

### Path Parameters

<ParamField path="id" type="string" required>
  Brand ID
</ParamField>

### Response

<ResponseField name="overall_score" type="number">
  Overall visibility score (0-100)
</ResponseField>

<ResponseField name="engine_scores" type="object">
  Scores per engine
  
  <Expandable title="Engine scores properties">
    <ResponseField name="chatgpt" type="number | null">
      ChatGPT visibility score
    </ResponseField>
    
    <ResponseField name="perplexity" type="number | null">
      Perplexity visibility score
    </ResponseField>
    
    <ResponseField name="google_aio" type="number | null">
      Google AI Overviews visibility score
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="sentiment" type="object">
  Sentiment distribution
  
  <Expandable title="Sentiment properties">
    <ResponseField name="positive" type="number">
      Percentage of positive mentions
    </ResponseField>
    
    <ResponseField name="neutral" type="number">
      Percentage of neutral mentions
    </ResponseField>
    
    <ResponseField name="negative" type="number">
      Percentage of negative mentions
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total_mentions" type="number">
  Total number of brand mentions
</ResponseField>

<ResponseField name="total_prompts" type="number">
  Total number of active prompts
</ResponseField>

<ResponseField name="recent_changes" type="Change[]">
  Array of recent changes
  
  <Expandable title="Change properties">
    <ResponseField name="id" type="string">
      Change ID
    </ResponseField>
    
    <ResponseField name="type" type="string">
      Change type: visibility_up, visibility_down, new_mention, sentiment_shift
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Human-readable description
    </ResponseField>
    
    <ResponseField name="engine" type="string | null">
      Engine where change occurred
    </ResponseField>
    
    <ResponseField name="change_value" type="number | null">
      Numeric change value
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl -X GET "https://api.opensight.ai/api/brands/brand_123/dashboard" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Example Response

```json

  "engine_scores": {
    "chatgpt": 75,
    "perplexity": 68,
    "google_aio": 73
  },
  "sentiment": {
    "positive": 65,
    "neutral": 30,
    "negative": 5
  },
  "total_mentions": 45,
  "total_prompts": 25,
  "recent_changes": [
    {
      "id": "change_123",
      "type": "visibility_up",
      "description": "Visibility increased by 5 points on ChatGPT",
      "engine": "chatgpt",
      "change_value": 5,
      "created_at": "2024-01-15T10:00:00Z"
    }
  ]
}
```

### Errors

- **404 Not Found** - Brand not found or user doesn't own it

## Get Brand Trends

<api method="GET" path="/api/brands/:id/trends" auth="required">
  Get trend data points for visibility scores over time.
</api>

### Path Parameters

<ParamField path="id" type="string" required>
  Brand ID
</ParamField>

### Query Parameters

<ParamField query="range" type="string">
  Time range: `7d`, `14d`, `30d`, or `90d` (default: 30d)
</ParamField>

### Response

<ResponseField name="trends" type="TrendDataPoint[]">
  Array of trend data points
  
  <Expandable title="TrendDataPoint properties">
    <ResponseField name="date" type="string">
      ISO 8601 date
    </ResponseField>
    
    <ResponseField name="overall" type="number">
      Overall visibility score
    </ResponseField>
    
    <ResponseField name="chatgpt" type="number | null">
      ChatGPT score
    </ResponseField>
    
    <ResponseField name="perplexity" type="number | null">
      Perplexity score
    </ResponseField>
    
    <ResponseField name="google_aio" type="number | null">
      Google AI Overviews score
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl -X GET "https://api.opensight.ai/api/brands/brand_123/trends?range=30d" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Example Response

```json
{
  "trends": [
    {
      "date": "2024-01-15",
      "overall": 72,
      "chatgpt": 75,
      "perplexity": 68,
      "google_aio": 73
    },
    {
      "date": "2024-01-16",
      "overall": 74,
      "chatgpt": 77,
      "perplexity": 70,
      "google_aio": 75
    }
  ]
}
```

### Errors

- **404 Not Found** - Brand not found or user doesn't own it
