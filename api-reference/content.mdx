## Score Content

<api method="POST" path="/api/content/score" auth="required">
  Score a URL for AI search engine optimization. Returns detailed scores across multiple dimensions with actionable recommendations. Daily limits apply based on your plan.
</api>

### Request Body

<ParamField body="url" type="string" required>
  Valid URL to score (must be a publicly accessible webpage)
</ParamField>

### Response

<ResponseField name="score" type="ContentScore">
  The content score result
  
  <Expandable title="ContentScore properties">
    <ResponseField name="id" type="string" required>
      Unique score identifier
    </ResponseField>
    
    <ResponseField name="user_id" type="string" required>
      User who requested the score
    </ResponseField>
    
    <ResponseField name="url" type="string" required>
      URL that was scored
    </ResponseField>
    
    <ResponseField name="overall_score" type="number" required>
      Overall content score (0-100)
    </ResponseField>
    
    <ResponseField name="structure_score" type="number | null">
      Structure and formatting score (0-100)
    </ResponseField>
    
    <ResponseField name="readability_score" type="number | null">
      Readability and clarity score (0-100)
    </ResponseField>
    
    <ResponseField name="freshness_score" type="number | null">
      Content freshness score (0-100)
    </ResponseField>
    
    <ResponseField name="key_content_score" type="number | null">
      Key content quality score (0-100)
    </ResponseField>
    
    <ResponseField name="citation_score" type="number | null">
      Citation and reference score (0-100)
    </ResponseField>
    
    <ResponseField name="recommendations" type="ContentRecommendation[]" required>
      Array of actionable recommendations
      
      <Expandable title="ContentRecommendation properties">
        <ResponseField name="dimension" type="string">
          Dimension of the recommendation (structure, readability, freshness, key_content, citation)
        </ResponseField>
        
        <ResponseField name="message" type="string">
          Human-readable recommendation message
        </ResponseField>
        
        <ResponseField name="priority" type="string">
          Priority level: high, medium, or low
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="scored_at" type="string" required>
      ISO 8601 timestamp when content was scored
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl -X POST "https://api.opensight.ai/api/content/score" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/blog/how-to-improve-seo"
  }'
```

### Example Response

```json
{
  "score": {
    "id": "score_123",
    "user_id": "user_456",
    "url": "https://example.com/blog/how-to-improve-seo",
    "overall_score": 78,
    "structure_score": 85,
    "readability_score": 72,
    "freshness_score": 90,
    "key_content_score": 75,
    "citation_score": 68,
    "recommendations": [
      {
        "dimension": "citation",
        "message": "Add more authoritative external citations to strengthen credibility",
        "priority": "high"
      },
      {
        "dimension": "readability",
        "message": "Break up long paragraphs for better readability",
        "priority": "medium"
      },
      {
        "dimension": "key_content",
        "message": "Include more specific examples and case studies",
        "priority": "medium"
      }
    ],

  }

```

### Errors

- **429 Too Many Requests** - Daily content scoring limit reached for your plan
- **400 Bad Request** - Invalid URL or URL is not accessible

### Rate Limits

- **Free**: 5 content scores per day
- **Starter**: 25 content scores per day
- **Growth**: 100 content scores per day

## Get Score History

<api method="GET" path="/api/content/score/history" auth="required">
  Get historical content scores for the authenticated user.
</api>

### Query Parameters

<ParamField query="limit" type="string">
  Number of results to return (default: 50, max: 50)
</ParamField>

<ParamField query="offset" type="string">
  Number of results to skip for pagination (default: 0)
</ParamField>

### Response

<ResponseField name="scores" type="ContentScore[]">
  Array of content score objects (see Score Content response for object structure)
</ResponseField>

<ResponseField name="total" type="number">
  Total number of scores in history
</ResponseField>

<ResponseField name="limit" type="number">
  Applied limit
</ResponseField>

<ResponseField name="offset" type="number">
  Applied offset
</ResponseField>

### Example Request

```bash
curl -X GET "https://api.opensight.ai/api/content/score/history?limit=10&offset=0" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
